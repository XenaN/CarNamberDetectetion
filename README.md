# CarNumberDetection

This project is experiment with yolo dists for detecting car numbers.

### Requirements

- Python >= 3.8
- RAM >= 16GB
- GPU >= 8GB
- CUDA >= 11.0

## Quick start

### Environment setup

Make executable file for environment setup:

```bash
chmod +x env_setup.sh
```

Run environment setup:

```bash
./env_setup.sh
```

### Usage with DVC

Run the following command to start the project:

```bash
dvc repro
```
